# Working time tracker (Wott)

Wott is web app written in PHP to track and store 
working hour.

## Dependencies

To use this app in the best conditions, please use :

- Php >= 7.4
- MySQL or MariaDB

Or

- Docker >= 18.0

## Installation

```bash
docker-compose up -d
```
